# Portal de Eventos 

Após clonar o repositório:

### Backend (API)
1. Ir até a pasta da API: `cd PortalEventos.Api`
2. Instalar as ferramentas do EF no terminal do Visual Studio: `dotnet tool install --global dotnet-ef`
3. Criar o banco de dados local: `dotnet ef database update`
4. Iniciar o servidor: `dotnet run`

### Frontend (Web)
1. Navegar até a pasta web: `cd portal-eventos-web`
2. Instalar as dependências: `npm install`
3. Iniciar o projeto: `npm run dev`
